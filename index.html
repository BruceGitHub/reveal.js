The Git bisect command

From the Git documentation:

This command uses a binary search algorithm to find which commit in your projectâ€™s history introduced a bug.

But what is a binary search algorithm?

Binary search algorithm

It searches for a value inside an ordered list/array

Easy example:
we are searching for number 3
inside the [1...9] interval


1 2 3 4 5 6 7 8 9

1 2 3 4 5 6 7 8 9

1 2 3 4 5 6 7 8 9

How binary search is used

With git bisect:

  • The list is the commit history of one branch
  • The order is time
  • Who decide if we are before or after? YOU!
  • HOW???
    • With a manual inspection
    • Refreshing a page
    • Using an automatic test!
